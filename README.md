Urban Threads Clothing Store Website

A modern and stylish e-commerce website for **Urban Threads**, a trendy boutique clothing brand specializing in affordable streetwear.  
This project was created to establish a strong online presence, improve brand identity, and expand the store’s market reach.


  Project Overview

Urban Threads is a small city-center boutique that offers fashionable yet affordable streetwear.  
This website provides an easy-to-use online platform where users can browse collections, view fashion tips, and contact the store.

The website was designed with a vibrant gradient color scheme (orange–yellow) to reflect the brand’s trendy and energetic identity.



 Website Objectives

- Expand the store’s market reach through online shopping  
- Provide an easy-to-use e-commerce interface  
- Build brand identity and customer loyalty  
- Share updates on new arrivals, promotions, and seasonal sales  



  Features & Functionality

 Home Page
- Hero section with background image and brand tagline  
- Overview of key features and links to other pages  

 Shop Page
- Grid layout for products  
- Product images, names, and prices  
- Hover animations for interactivity  

Lookbook / Blog Page
- Style inspiration and fashion articles  
- Image-based layout showcasing streetwear looks  

 About Page
- Information about the Urban Threads brand  
- Brand story and mission statement  

  Contact Page
- Contact form for customer inquiries  
- Integrated email form, phone, and live chat section  


  Design & Aesthetic

- Color Theme: Orange (#ff7b00), Yellow (#ffbb00), and subtle white accents  
- Font: Poppins (Google Font)  
- Background: Gradient and optional image overlay  
- Layout: Modern, responsive grid-based design  
- Effects:Hover transitions, shadows, rounded corners
